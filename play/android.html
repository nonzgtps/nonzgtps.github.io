How to play Zeetopia
  • Android
  • Ios
  • MacOs
  • Windows

Host go

1. Download "(No Root) Hosts GO".

Download Host GO

2. Install "(Not Root) Hosts GO" to your device.

3. Open Host GO and Click on "Hosts Editor".

4. Click checkbox on the right, and Click "Download Hosts File".

5. Type the link on bellow: "http://zeetopia.pages.dev/android".

6. Click on "Download and Apply".

7. Back and click on "START".

8. Open Growtopia and Connect!


Virtual Host

1. Download Virtual Hosts.

Download Virtual Host

2. Install Virtual Hosts.

3. Download Hosts File.

Download Host

4. Open Virtual Hosts and click on "SELECT HOST FILE".

5. Add the hosts file you downloaded.

6. Click on the big white checkbox and make it green.

7. Open Growtopia and connect!
